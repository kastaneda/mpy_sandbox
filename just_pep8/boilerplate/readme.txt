Firmware files naming scheme:

 - app.py
 - boot.py
 - config.py
 - ...
 - main.py
 - readme.txt

Rules:

 - all file names must be lowercase and fit 8.3 length
 - all files must start with different letters
 - <app.py> shall be the first file
 - <main.py> shall be the last executable script
 - <readme.txt> is optional (URL to GitHub may be enough)
